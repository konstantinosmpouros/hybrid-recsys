# data/

This directory holds all dataset files. Neither subdirectory is tracked by git.

## raw/

Place the **MovieLens 25M** CSV files here before running any notebook.

Download: [MovieLens 25M](https://grouplens.org/datasets/movielens/25m/)

Expected files:

| File | Description |
| --- | --- |
| `ratings.csv` | 25 M user–movie ratings (userId, movieId, rating, timestamp) |
| `movies.csv` | Movie titles and pipe-separated genres |
| `tags.csv` | Free-text tags applied by users |
| `genome-scores.csv` | Tag relevance scores for ~13 K movies × 1,128 tags |
| `genome-tags.csv` | Tag id → tag string mapping |
| `links.csv` | MovieLens → IMDb / TMDb id crosswalk (not used in modelling) |

## processed/

Auto-generated by `notebooks/01_eda.ipynb` and `notebooks/02_features.ipynb`.
Do not edit these files manually.

| File | Generated by | Content |
| --- | --- | --- |
| `movies.parquet` | 01_eda | Enriched movie table (title, year, genres, tags_text) |
| `ratings.parquet` | 01_eda | Cleaned ratings with consistent dtypes |
| `split_train.parquet` | 01_eda | 80 % temporal training split |
| `split_val.parquet` | 01_eda | 10 % validation split |
| `split_test.parquet` | 01_eda | 10 % test split (evaluated once) |
| `item_features.npz` | 02_features | Sparse item feature matrix (genre + TF-IDF/LSA) |
| `movie_index.parquet` | 02_features | movieId → row index mapping for item_features |
